# Basic Network Sniffer

### CodeAlpha Cyber Security Internship - Task 1

A simple Python-based network packet sniffer developed as part of the
**CodeAlpha Cyber Security Internship**. This project captures live
network traffic and displays essential packet information such as source
IP address, destination IP address, and protocol type.

------------------------------------------------------------------------

## 📌 Project Overview

The objective of this project is to understand how data travels across a
network by capturing and analyzing packets in real time.

The program uses the **Scapy** library to: - Capture live network
packets. - Identify the source and destination IP addresses. - Detect
common protocols (TCP, UDP, ICMP). - Display packet details in a simple
and readable format.

------------------------------------------------------------------------

## 🚀 Features

-   Live packet capturing
-   Source IP detection
-   Destination IP detection
-   Protocol identification
-   Lightweight and beginner-friendly
-   Built using Python and Scapy

------------------------------------------------------------------------

## 🛠️ Technologies Used

## ⚙️ Installation

### Clone the repository

``` bash
git clone https://github.com/YourUsername/CodeAlpha_BasicNetworkSniffer.git
```

### Navigate into the folder

``` bash
cd CodeAlpha_BasicNetworkSniffer
```

### Install dependencies

``` bash
pip install scapy
```

------------------------------------------------------------------------

## ▶️ Usage

Run the script:

``` bash
python network_sniffer.py
```

> **Note:** On Windows, run the terminal as Administrator for packet
> capture permissions.
